Тобі дано довгий рядок (рівномірний шрифт), і ти повинен розділити його на рядки із урахуванням максимальної ширини. Також необхідно відформатувати текст відповідно до заданого стилю: "l" означає вирівнювання тексту по лівому краю (left), "c" - по центру (center), "r" - по правому краю (right), і "j" означає вирівнювання по ширині (justify). Нарешті, рядки в результаті не повинні закінчуватися пробілами.

Якщо тобі доведеться поставити загалом непарну кількість пробілів навколо рядка, щоб розмістити його посередині, тоді з початку рядка розміщую парну кількість пробілів.

Правила вирівнювання по ширині:

  • оскільки ми не завжди можемо поставити рівну кількість пробілів між словами в рядку, спочатку став більшу кількість пробілів стільки разів, скільки це необхідно. Наприклад: X---X---X--X--X--X коли тобі потрібно розмістити 12 пробілів у 5 проміжках між словами: 3-3-2-2-2;
  • НЕ вирівнюй по ширині останній рядок тексту.

Тобі не знадобиться ділити слово на дві частини, оскільки задана ширина досить велика.
