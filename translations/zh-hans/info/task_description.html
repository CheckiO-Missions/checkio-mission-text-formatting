给您一个长行（monospace字体），您必须按照给定的宽度断行。然后，您必须根据给定的样式对文本进行格式化："l" 表示文本必须向左对齐，"c" 表示居中对齐，"r" 表示向右 对齐，"j" 表示文本必须对齐。最后，输出结果的行尾不能有空格。

如果您必须在一行周围空 2 * n + 1 个空格才能使其居中，那就在前面空 n 个空格，而不是 n + 1 个空格。

说明理由规则：

  • 由于我们不能总是在一行中的单词之间放置相同数量的空格，因此应先放置大的空格块。例如X---X---X---X---X----X，当您必须在5个空格中填入12个空格时：3-3-2-2-2;
  • 不要为文本的最后一行对齐。

您不必考虑将一个单词分成两部分，因为给定的宽度已经足够大了。

输入：一个文本 (str{% if is_js %}ing{% endif %})，宽度 ({% if is_js %}number{% else %}int{% endif %})和样式 (str{% if is_js %}ing{% endif %}).

输出：格式化文本 ({% if is_js %}string{% else %}str{% endif %})。
