Sie erhalten eine lange Zeile (eine Monospace-Schriftart) und müssen die Zeile so umbrechen, dass eine bestimmte Breite eingehalten wird. Dann müssen Sie den Text entsprechend dem vorgegebenen Stil formatieren: "l" bedeutet, dass Sie den Text linksausrichten müssen, "c" für die Mitte, "r" für rechtsund "j" bedeutet, dass Sie den Text im Blocksatz ausrichten müssen. Schließlich sollten die Zeilen der Ausgabe nicht mit einem Leerzeichen enden.

Wenn Sie 2 * n + 1 Leerzeichen um eine Zeile setzen müssen, um sie zu zentrieren, dann setzen Sie n Leerzeichen davor, nicht n + 1.

Die Rechtschreibregeln:

  • da wir nicht immer die gleiche Anzahl von Leerzeichen zwischen die Wörter in einer Zeile setzen können, setzen Sie große Blöcke von Leerzeichen vor. Zum Beispiel: X---X---X--X--X--X, wenn Sie 12 Leerzeichen in 5 Lücken setzen müssen: 3-3-2-2-2;
  • richten Sie die letzte Zeile eines Textes nicht aus.

Sie müssen nicht in Betracht ziehen, ein Wort in zwei Teile aufzuteilen, da die angegebenen Breiten groß genug sind.
